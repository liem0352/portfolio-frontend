# 个人作品集网站

> 作者：liem

## 项目简介

个人作品集前端网站，使用纯 HTML + CSS + JavaScript 构建，包含多个页面模块，展示个人项目与技能。

## 功能页面

- 首页（index）：个人介绍与项目展示
- 博客（blog）：技术博客文章列表
- 简历（resume）：个人简历与教育经历
- 联系方式（contact）：联系方式与留言
- 首页（home）：综合展示
- Markdown 展示（showmd）：在线 Markdown 渲染

## 技术栈

- HTML5
- CSS3（动画、Flexbox 布局）
- 原生 JavaScript
- Prism.js 代码高亮
- Marked.js Markdown 解析
- Canvas 粒子特效
- QRCode 二维码生成

## 使用方式

直接在浏览器中打开 `index.html` 即可预览。
